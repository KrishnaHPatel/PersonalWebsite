Courses

Here is an overview of the relevant courses I have taken at Cornell! I have been on the Dean's List for Academic Excellence for all semesters at Cornell.

CS 2110: Object-Oriented Programming & Data Structures

Learned intermediate programming in Java, including object-oriented principles (classes, inheritance, polymorphism), algorithm analysis, recursion, and various data structures such as lists, trees, and hash tables. Gained experience in building event-driven GUIs and understanding concurrent programming.

CS 2800: Discrete Structures

Covered essential mathematical concepts for computer science, such as logical proofs, set theory, combinatorics, graph theory, and basic probability. Applied these concepts to various computer science problems.

CS 3110: Data Structures & Functional Programming

Focused on advanced programming with an emphasis on functional programming techniques, recursive and higher-order procedures, and data structures like balanced trees and heaps. Analyzed program performance and correctness.

CS 4320: Introduction to Database Systems

Explored modern database systems, covering data models, query languages, database design, storage structures, query processing, and transaction management in both relational and non-relational databases.

CS 4780: Introduction to Machine Learning

Introduced to machine learning concepts with a focus on supervised learning. Studied linear models, deep networks, boosting, and ethical considerations in ML applications.

CS 4820: Introduction to the Analysis of Algorithms

Developed techniques for designing and analyzing algorithms, with topics including greedy algorithms, dynamic programming, NP-completeness, and approximation algorithms. Applied these techniques to real-world computing problems.

MATH 1920: Multivariable Calculus

Learned about partial derivatives, multiple integrals, vector fields, and important theorems like Green's, Stokes', and the divergence theorem, which are fundamental for understanding physical phenomena and engineering problems.

MATH 2930: Differential Equations

Studied both ordinary and partial differential equations, covering methods for solving them and applications in modeling physical systems, such as population dynamics and heat conduction.

MATH 2940: Linear Algebra

Focused on matrices, determinants, vector spaces, eigenvalues, and eigenvectors, with applications to Markov chains and linear differential equations. Explored the use of linear algebra in computational problems.

PHYS 1112: Mechanics & Heat

Covered fundamental concepts in mechanics, including kinematics, dynamics, conservation laws, and thermodynamics. Emphasized understanding through interactive and problem-solving approaches.

PHYS 2213: Electromagnetism

Explored electric and magnetic forces, fields, potentials, and circuits, along with Maxwell's equations. Applied advanced mathematical techniques like vector calculus to solve physics problems.
