Production-Level Projects

Two major projects delivered from concept to production

Project 1: Prompt Sharing of Databases

Goal

Increase visibility and collaboration on Confluence databases by encouraging users to share new databases shortly after creation.

Solution

Built a frontend feature that detects when a database is newly created and triggers a share prompt within 5 minutes. Developed custom React hooks (useDatabaseQueryParamState, useShareNewDatabaseFlag) and integrated with experimentation frameworks.

Impact

Targeted a +7% lift in database engagement; drove metric selection via 4+ pre-analyses in Databricks/Socrates.

Project 2: AI-Generated "What's New" Content

Goal

Automate the monthly "What's New in Confluence" homepage updates, which were previously manually curated from digest emails.

Solution

Designed and deployed a microservice (controller/service/data layers) leveraging AI Gateway SDK (GPT-4), Confluence REST APIs, and AWS S3 to summarize and publish the latest feature updates. Integrated with the frontend via StarGate for real-time publishing.

Impact

Reduced manual update effort by ~90% and enabled publishing to tens of thousands of users with minimal human intervention.
