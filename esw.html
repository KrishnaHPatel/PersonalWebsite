Project Overview

Leading a team to build a sustainable food waste reduction platform

I lead a team of 5 full-stack developers to build a web platform aimed at serving 500+ student organizations, developing features using the MERN tech stack to enable posting and retrieval of excess food, aimed at reducing food waste.

Our web application will allow users to view posts, create new posts, and search through existing posts. It will be restricted to Cornell students via Google Auth and deployed on Netlify/Render.

Key Features

Core functionality of the food waste reduction platform

View Posts

Browse through existing food posts to see what's available and when it was posted.

Create New Posts

Post excess food items with details like quantity, location, and expiration time.

Search Functionality

Search through posts by keywords, location, or food type to find what you need.
